Introduction to Android

Goals

  1. Create and run your first Android app.
  2. Learn how to use a ListView to show data.
  3. Learn how to add a button to the ActionBar.
  4. Learn how to launch a simple dialog.
  5. Learn the easiest way to store data with Android SQLite.
  6. Learn how to make HTTP reuqests to a server.

Lab 4 Import the Language Helper code

  1. In Eclipse, go to File | Import | Android > Import Existing Android Code into Workspace
  2. Choose the language-helper-arequipa folder
  3. Click Finish. You should see the project created with no errors.
  4. Run the app (Run As | Android Application).

Lab 5 Work with a ListView

  1. Locate the file ExpandableListAdapter.java. See method getChildView().
  2. Change the layout files so that the Spanish word always appears in red (hint: use the android:textColor attribute).
  3. See method getGroupView().
  4. Modify the appropriate layout file so that the group headings appear in bold text.

Lab 6 Add a button to the ActionBar

  1. Locate onCreateOptionsMenu in MainActivity.java.
  2. In the corresponding layout file, comment out the add button.
  3. Note the error which now appears in the Java code and comment it out, too.
  4. Run the app to see that the button is gone.
  5. Uncomment the lines you commented out to restore the button.

Lab 7 Replace an icon on the ActionBar

  1. Go to File | New | Android Icon Set.
  2. Choose Action Bar and Tab Icons.
  3. Name the icon ic_action_add to match the existing name. Click Next.
  4. Choose a new image from Clipart. Click Next, then Replace All when you get the warning.
  5. Run the app to see the new button in the action bar.
