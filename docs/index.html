Introduction to Android

Goals

  1. Create and run your first Android app.
  2. Learn how to use a ListView to show data.
  3. Learn how to add a button to the ActionBar.
  4. Learn how to launch a simple dialog.
  5. Learn the easiest way to store data with Android SQLite.
  6. How to wrap Web content with a WebView.

Lab 9 Use a WebView

  1. File | New | Android Layout XML File
  2. Choose Layout XML File. Click Next.
  3. In the File: field, type layout_webview
  4. In the Root Element field, choose WebView. Click Finish.
  5. When the new file appears, click the word WebView in the center of the design. In the Properties panel to the right, click the three dots for the Id attribute (first in the list) and type mywebview. The XML code for layout_webview.xml should look like the snippet below.
  6. In MainActivity.java, create a new method showWebView() using the code below.
  7. Locate onCreateOptionsMenu() in MainActivity.java.
  8. In the case for R.id.action_toast, replace the code to show a Toast with the new method showWebView().
  9. In AndroidManifest.xml, add the required permission shown below.
  10. Run the app to see that the button loads the Google homepage in your app.
layout_webview.xml: 
<?xml version="1.0" encoding="utf-8"?>
  <WebView xmlns:android="http://schemas.android.com/apk/res/android"
  android:id="@+id/mywebview"
  android:layout_width="match_parent"
  android:layout_height="match_parent" >
</WebView>
MainActivity.java: 
	private void showWebView() {
		setContentView(R.layout.layout_webivew);
		WebView webview = (WebView) findViewById(R.id.mywebview);
		webview.setWebViewClient(new WebViewClient());
		webview.loadUrl("http://www.google.com");
	}
AndroidManifest.xml: 
    <uses-permission android:name="android.permission.INTERNET" />

Lab 10 Use storm-gen and write a simple unit test

In this exercise, you are going to use storm-gen to access the Android SQLite database from a unit test.

  1. File | New | Android Test Project. Name the new project lhtest. Click Next and select the language helper project as the test target. Click Finish.
  2. In the test project, create a new class StormTest that extends AndroidTestCase.
  3. Create a new method public void testQuery().
  4. In the method, create a new PalabraDao(getContext());
  5. Use dao.load.list() to retrieve all Palabra records.
  6. Write an assertion (assertEquals) that says there should be 96 records.
  7. Run the test project as an Android Unit Test. See that it fails.
  8. Now change the assertion to 110 records and run it again. It should pass.
